# 辞書と世界 (Dictionary & The World)

SmartIMEにおいて「辞書」とは、単なる文字変換のためのリストではありません。それは、ユーザーが生き、思考し、表現する「世界の広さ」そのものを定義する基盤です。

## 1. 辞書は「世界の境界線」である
- 言葉を知らなければ、その事象を思考することはできません。IMEの辞書が貧弱であることは、ユーザーの思考の翼を折ることに等しい。
- SmartIMEは、標準的な語彙を超え、プログラムコード、専門用語、ネットスラング、そしてユーザー独自の「私的な言葉」を等しく受け入れ、拡張し続けます。

## 2. 公共の言葉と、私的な庭 (Personal Delta)
- **公共の辞書**: 社会が共有する言語体系。これは「共通の基盤」として機能します。
- **私的な庭**: ユーザー独自の言いまわし、特殊な略語、変換の癖。SmartIMEは、これらを「Personal Delta」としてローカルに蓄積します。
- システムは、公共の言葉をベースとしながらも、ユーザー個人の「世界（私的言語）」を最優先して理解します。

## 3. 癖（癖）という名のショートカット
- 打ち間違いや独特の省略は、エラーではなく「進化の兆し」です。
- ユーザーが `npm i` と打てば、それは `npm install` へのショートカットであると同時に、開発という「世界」に没入している証拠です。
- SmartIMEは、ユーザーの「癖」を学習・圧縮し、思考と入力の距離を極限まで縮めます。

## 4. テーマ辞書による「世界」の切り替え
- 開発者の世界、医師の世界、ゲーマーの世界。
- 手動でモードを切り替えるのではなく、入力された数文字の「メタデータ」から、今ユーザーがどの「世界」の言葉を使おうとしているのかを自律的に察知し、辞書の重み付けを動的に変更します。

## 5. 辞書なき後の「純粋思考」
- 最終的な目標は、辞書の存在さえ意識させないことです。
- ユーザーが何かを伝えたい、あるいはコードを書きたいと思ったとき、辞書は空気のように背後に徹し、あたかも脳から直接文字が溢れ出しているかのような体験を提供します。
- 言葉を選ぶストレスから解放されたとき、人間は初めて「純粋な思考」に向き合うことができます。
